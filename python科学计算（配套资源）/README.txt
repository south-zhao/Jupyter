●本书样本的操作环境
　本书以Windows10 (64bit)环境为基础进行解说。使用Anaconda Individual Edition (Anaconda3-2020.02- windows -x86_64)来安装Python和库。

・Anaconda Individual Edition
https://www.anaconda.com/distribution/

・Anaconda Installer archive（如果在上述网站没有，请从以下网站下载）。
https://repo.anaconda.com/archive/
